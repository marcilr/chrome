Disable saving passwords in chrome


1. Type the following into the browser:
chrome://settings (You could also get there using the chrome menu, top right)

2. Then type: passwords (in the search box on the top right of the settings
   page)

3. Then deselect/unclick the option "Offer to save your web passwords."
